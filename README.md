# 📚 Livraria em Java

Sistema simples de livraria desenvolvido em Java utilizando conceitos de **Programação Orientada a Objetos (POO)**.
O projeto simula o funcionamento básico de uma biblioteca, incluindo gerenciamento de livros, usuários e empréstimos.

---

## 🚀 Funcionalidades

* 📖 Cadastro e exibição de livros
* 👤 Cadastro de usuários e membros
* ✍️ Cadastro de autores
* 📦 Controle de itens de livros
* 🔄 Empréstimo de livros
* 💰 Cálculo de multa por atraso
* 🔍 Busca, adição e remoção de livros na biblioteca
* 🧑‍💼 Gerenciamento de acervo por bibliotecário
* 🔁 Renovação de matrícula de membros

---

## 🛠️ Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* NetBeans

---

## 📂 Estrutura do Projeto

O sistema é composto pelas seguintes classes:

* `Livraria` → Classe principal (main)
* `Livro` → Representa um livro
* `Autor` → Informações sobre o autor
* `Usuario` → Dados do usuário do sistema
* `Membro` → Usuário com vínculo à biblioteca
* `Bibliotecario` → Funcionário responsável pelo acervo
* `Biblioteca` → Gerenciamento dos livros
* `ItemLivro` → Controle físico do livro (status, código)
* `Emprestimo` → Controle de empréstimos e multas

---

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/livraria-java.git
```

2. Abra o projeto no NetBeans

3. Execute a classe `Livraria.java`

## 💡 Exemplo de execução

O sistema cria objetos automaticamente e exibe informações como:

* Dados dos livros
* Informações dos usuários
* Status dos itens
* Cálculo de multas
* Operações da biblioteca (adicionar, buscar, remover livros)
